## 使用示例

### 基本配置访问
```python
from day_translation.utils.unified_config import get_config

config = get_config()
print(f"默认语言: {config.core.default_language}")
print(f"自动模式: {config.user.general.auto_mode}")
```

### 路径管理
```python
# 记住路径
config.remember_path("mod_dir", "/path/to/mod")

# 获取记住的路径
mod_path = config.get_remembered_path("mod_dir")

# 智能路径输入
path = config.get_path_with_validation(
    path_type="mod_dir",
    prompt="请输入模组目录",
    validator_type="mod"
)
```

### 智能输出目录选择
```python
from day_translation.utils.unified_interaction_manager import UnifiedInteractionManager

manager = UnifiedInteractionManager()
result = manager._get_smart_output_directory("输出目录", "默认路径")
if result:
    output_dir, processing_mode = result
    print(f"目录: {output_dir}, 处理方式: {processing_mode}")
```
