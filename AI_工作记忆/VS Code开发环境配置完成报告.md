## 🛠️ 开发工作流程

### 日常开发任务
- `Ctrl+Shift+P` → `Tasks: Run Task` → 选择需要的任务
- 或使用快捷键：
  - `Ctrl+Shift+T` - 运行测试
  - `Ctrl+Shift+B` - 构建/格式化

### 可用的VS Code任务
1. **测试相关**
   - `pytest: 运行所有测试` - 执行完整测试套件
   - `pytest: 测试覆盖率报告` - 生成覆盖率报告

2. **代码质量**
   - `black: 格式化代码` - 统一代码格式
   - `flake8: 静态检查` - 代码风格检查
   - `mypy: 类型检查` - 类型安全检查
   - `质量检查: 完整检查` - 运行所有质量检查

3. **环境管理**
   - `环境检查` - 验证Python环境和依赖

### 调试配置
- 当前文件调试 - 调试正在编辑的Python文件
- 项目主程序调试 - 调试Day_translation2或Day_EN主程序
- 测试调试 - 调试单个测试文件或整个测试套件
