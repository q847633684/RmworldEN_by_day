**决策时间**: 2024年当前会话

**架构设计理念**:

### ADR-022: 统一OperationResult返回模型
**决策时间**: 2024年当前会话
**决策内容**: 所有业务函数统一返回OperationResult类型

**统一返回格式**:
```python
@dataclass
class OperationResult:
    status: OperationStatus          # 操作状态
    operation_type: OperationType    # 操作类型
    message: str                     # 结果消息
    processed_count: int = 0         # 处理数量
    success_count: int = 0          # 成功数量
    error_count: int = 0            # 错误数量
    details: List[str] = field(default_factory=list)  # 详细信息
    execution_time: Optional[float] = None            # 执行时间
```

**决策理由**:
- **一致性**: 所有函数返回格式统一
- **信息丰富**: 包含完整的操作结果信息
- **易于处理**: 调用方可统一处理结果
- **错误友好**: 详细的错误信息和上下文

**替代旧系统**:
- 旧系统: 返回tuple、bool、dict等不一致格式
- 新系统: 统一OperationResult格式

**状态**: ✅ 模型定义完成，开始应用
