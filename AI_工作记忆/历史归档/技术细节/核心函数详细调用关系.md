### 1. 核心提取器调用链
```
extract_keyed_translations / extract_definjected_translations
├── AdvancedFilterRules.should_translate_*
├── AdvancedXMLProcessor
├── TranslationEntry (数据模型创建)
└── get_language_folder_path (路径处理)

scan_defs_sync
├── _extract_translatable_fields_recursive (递归核心)
├── ThreadPoolExecutor (并发处理)
└── tqdm (进度显示)
```

### 2. 导入器调用链
```
import_translations
├── load_translations_from_csv (CSV处理)
└── update_all_xml (XML批量更新)
    ├── XMLProcessor (XML操作)
    ├── shutil.copy2 (备份)
    └── get_language_folder_path (路径处理)
```

### 3. 导出器调用链
```
export_keyed / export_definjected
├── _generate_*_xml (XML生成)
├── ExportError (异常处理)
├── OperationResult (结果封装)
└── get_language_folder_path (路径处理)

export_with_smart_merge
├── ExportManager (导出管理)
├── ExportMode (模式控制)
└── export_manager.export_* (具体导出)
```

### 4. 递归调用识别
- `_extract_translatable_fields_recursive` 存在自递归调用
- 用于遍历XML元素树结构
- 是系统中最深层的递归逻辑

### 5. 并发处理识别
- `scan_defs_sync` 使用 `ThreadPoolExecutor`
- 提供并发XML文件处理能力
- 配合 `tqdm` 提供进度反馈
