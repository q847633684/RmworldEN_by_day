## 🚨 常见问题快速诊断

### 类别 1: 导入错误 (ImportError)

#### 问题 1.1: 模块未找到
**症状：**
```
ModuleNotFoundError: No module named 'day_translation.utils.unified_config'
```

**可能原因：**
- [ ] 文件路径错误
- [ ] `__init__.py` 文件缺失
- [ ] 相对导入路径错误

**排查步骤：**
1. 检查文件是否存在：`ls day_translation/utils/unified_config.py`
2. 检查 `__init__.py`：`ls day_translation/__init__.py`
3. 检查相对路径：从当前文件位置计算正确的相对路径

**解决方案：**
```python
# 错误的导入
from day_translation.utils.unified_config import get_config

# 正确的导入（根据实际路径调整）
from ..utils.unified_config import get_config
# 或者
from day_translation2.config.unified_config import get_config
```

#### 问题 1.2: 循环导入
**症状：**
```
ImportError: cannot import name 'get_config' from partially initialized module
```

**可能原因：**
- [ ] 模块A导入模块B，模块B也导入模块A
- [ ] 导入语句在函数外部

**排查步骤：**
1. 绘制导入依赖图
2. 查找循环依赖路径
3. 检查是否可以延迟导入

**解决方案：**
```python
# 延迟导入
def get_interaction_manager():
    from ..interaction.unified_interaction import UnifiedInteractionManager
    return UnifiedInteractionManager()

# 或者重新设计模块结构
```

### 类别 2: 方法调用错误 (AttributeError)

#### 问题 2.1: 方法不存在
**症状：**
```
AttributeError: 'UnifiedInteractionManager' object has no attribute 'handle_preferences_menu'
```

**可能原因：**
- [ ] 方法名称错误
- [ ] 方法已被重命名或移除
- [ ] 使用了错误的类实例

**排查步骤：**
1. 检查类定义：`grep -n "class UnifiedInteractionManager" *.py`
2. 查找方法定义：`grep -n "def handle_" *.py`
3. 检查方法重命名记录

**解决方案：**
```python
# 错误调用
interaction_manager.handle_preferences_menu()

# 正确调用
interaction_manager.handle_settings_menu()
```

#### 问题 2.2: 方法签名不匹配
**症状：**
```
TypeError: show_operation_result() takes 3 positional arguments but 4 were given
```

**可能原因：**
- [ ] 方法参数数量不匹配
- [ ] 参数类型不匹配
- [ ] 缺少可选参数

**排查步骤：**
1. 查看方法定义：`grep -A 5 "def show_operation_result" *.py`
2. 检查调用处的参数：`grep -B 2 -A 2 "show_operation_result(" *.py`
3. 对比期望签名和实际签名

**解决方案：**
```python
# 检查当前方法签名
def show_operation_result(self, success: bool, message: str):
    # 原始实现

# 修改为支持详细信息的签名
def show_operation_result(self, success: bool, message: str, details: List[str] = None):
    # 增强实现
```

**症状：**
```

```

**可能原因：**
- [ ] JSON 格式错误
- [ ] 配置文件损坏

**排查步骤：**
1. 验证 JSON 格式：`python -m json.tool config.json`
2. 检查文件编码：`file -I config.json`

**解决方案：**

try:
    with open(config_path, 'r', encoding='utf-8') as f:

except json.JSONDecodeError as e:
    logging.error(f"配置文件格式错误: {e}")

#### 问题 3.2: 配置项缺失
**症状：**
```
KeyError: 'default_language'
AttributeError: 'CoreConfig' object has no attribute 'definjected_dir'
```

**可能原因：**
- [ ] 配置文件版本过旧
- [ ] 配置项名称变更
- [ ] 默认值未设置

**排查步骤：**
1. 检查配置文件版本
2. 对比新旧配置结构
3. 查看默认值设置

**解决方案：**
```python
# 兼容旧配置
@property
def definjected_dir(self):
    # 兼容旧的命名
    return getattr(self, 'def_injected_dir', 'DefInjected')

# 或者配置迁移
def migrate_config(old_config):
    new_config = {}
    new_config['definjected_dir'] = old_config.get('def_injected_dir', 'DefInjected')
    return new_config
```

### 类别 4: 路径相关错误

#### 问题 4.1: 路径不存在
**症状：**
```
FileNotFoundError: [Errno 2] No such file or directory: '/path/to/mod'
```

**可能原因：**
- [ ] 路径字符串错误
- [ ] 文件或目录被删除
- [ ] 权限问题

**排查步骤：**
1. 检查路径是否存在：`ls -la /path/to/mod`
2. 检查权限：`ls -ld /path/to/mod`
3. 验证路径字符串构造

**解决方案：**
```python
# 添加路径验证
def validate_mod_directory(path):
    if not os.path.exists(path):
        raise FileNotFoundError(f"模组目录不存在: {path}")
    if not os.path.isdir(path):
        raise NotADirectoryError(f"路径不是目录: {path}")
    return path
```

#### 问题 4.2: 路径分隔符问题
**症状：**
- Windows 上路径包含 `/`
- Unix 上路径包含 `\`

**解决方案：**
```python
# 使用 pathlib 处理路径
from pathlib import Path

path = Path(user_input_path)
normalized_path = path.resolve()

# 或者使用 os.path
import os
normalized_path = os.path.normpath(user_input_path)
```

### 类别 5: 数据处理错误

#### 问题 5.1: CSV 文件格式错误
**症状：**
```
pandas.errors.EmptyDataError: No columns to parse from file
```

**可能原因：**
- [ ] CSV 文件为空
- [ ] 编码问题
- [ ] 分隔符不正确

**排查步骤：**
1. 检查文件大小：`ls -la file.csv`
2. 查看文件内容：`head -5 file.csv`
3. 检查编码：`file -I file.csv`

**解决方案：**
```python
# 健壮的 CSV 读取
def load_csv_safely(csv_path):
    try:
        # 尝试不同编码
        for encoding in ['utf-8', 'gbk', 'latin-1']:
            try:
                df = pd.read_csv(csv_path, encoding=encoding)
                if not df.empty:
                    return df
            except UnicodeDecodeError:
                continue
        raise ValueError(f"无法读取 CSV 文件: {csv_path}")
    except Exception as e:
        logging.error(f"CSV 读取错误: {e}")
        return None
```

### 2. Python 调试代码

#### 方法存在性检查
```python
def check_method_exists(obj, method_name):
    """检查对象是否有指定方法"""
    if hasattr(obj, method_name):
        method = getattr(obj, method_name)
        if callable(method):
            print(f"✅ {method_name} 方法存在且可调用")
            import inspect
            sig = inspect.signature(method)
            print(f"   签名: {sig}")
        else:
            print(f"❌ {method_name} 存在但不可调用")
    else:
        print(f"❌ {method_name} 方法不存在")
        print(f"   可用方法: {[m for m in dir(obj) if not m.startswith('_')]}")
```

#### 配置检查代码

## 🆘 紧急处理流程

### 如果系统完全无法启动
1. **立即回滚** - 恢复最后一个可工作版本
2. **问题定位** - 查看错误日志和堆栈信息
3. **最小修复** - 只修复阻止启动的关键问题
4. **重新测试** - 确保基本功能可用

### 如果部分功能异常
1. **隔离问题** - 确定影响范围
2. **临时绕过** - 提供替代方案
3. **根因分析** - 使用调试工具定位原因
4. **精确修复** - 针对性解决问题
