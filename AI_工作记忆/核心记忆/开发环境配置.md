# Day汉化项目 - 开发环境配置

## 🎯 配置目标
为Day_translation2新系统建立完整的开发环境，确保代码质量和开发效率。

## 🐍 Python环境配置

### 基础环境
- **Python版本**: 3.8+ (推荐3.9+)
- **包管理**: pip + requirements.txt
- **虚拟环境**: venv (推荐) 或 conda

### 环境建立
```bash
# 创建虚拟环境
python -m venv day_translation2_env

# 激活环境 (Windows)
day_translation2_env\Scripts\activate

# 激活环境 (Linux/Mac)
source day_translation2_env/bin/activate

# 安装基础依赖
pip install -r requirements.txt
```
