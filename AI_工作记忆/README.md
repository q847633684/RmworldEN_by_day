## 🎯 使用指南 (最新工作流程)

### 快速导航
- **项目状态查询** → [快速状态.md](核心记忆/快速状态.md)
- **开发问题解决** → [问题排查手册.md](核心记忆/问题排查手册.md)
- **技术决策查阅** → [关键决策记录.md](核心记忆/关键决策记录.md)
- **质量检查状态** → [Pylint错误修复完整报告.md](核心记忆/Pylint错误修复完整报告.md)
- **类型检查优化** → [Mypy类型检查渐进式优化报告.md](核心记忆/Mypy类型检查渐进式优化报告.md)

### 每日工作流程
1. **开始工作** → 查看 `核心记忆/快速状态.md` 获取最新状态
2. **遇到问题** → 优先查询 `核心记忆/问题排查手册.md`
3. **质量检查** → 运行 `quality_check.py` (Day_translation2目录)
4. **重要决策** → 记录到 `核心记忆/关键决策记录.md`
5. **状态更新** → 更新 `核心记忆/快速状态.md`

### 信息查找策略
- **当前状态**: 优先查看核心记忆/快速状态.md
- **历史分析**: 查阅历史归档/对应主题目录
- **工具使用**: 参考核心记忆中的配置和优化文件
- **问题排查**: 按问题类型查询相应的报告文件
